# Leveraging Machine Learning and Transformer-Based Models for Social Media Sentiment Analysis in Brand Management

## Project Summary

This project investigates the effectiveness of Machine Learning, Deep Learning, and Transformer-based models for sentiment classification of airline customer feedback on social media.

Using the Twitter US Airlines Sentiment dataset, multiple approaches including SVM, Random Forest, XGBoost, CNN, LSTM, and BERT were trained and evaluated. The study compares model performance using accuracy, precision, recall, and F1-score to identify the most effective techniques for sentiment analysis and brand management applications.

## Technologies Used

- Python
- Scikit-learn
- TensorFlow / Keras
- Hugging Face Transformers
- XGBoost
- Pandas & NumPy
- NLTK

## Models Evaluated

- Support Vector Machine (SVM)
- Random Forest
- XGBoost
- K-Nearest Neighbours (KNN)
- Decision Tree
- AdaBoost
- CNN
- LSTM
- BERT

## Key Outcome

- BERT achieved the strongest overall performance.
- XGBoost delivered the best performance among traditional machine learning models.
- Comparative analysis highlighted the advantages of transformer architectures for sentiment classification tasks.
